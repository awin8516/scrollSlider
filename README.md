## script
###1.use javascript
```javascript
var myScrollSlider1 = new scrollSlider('scrollSlider_1', {
    direction : 'horizontal',//vertical 滚动方向 水平/垂直
    mode : 'left',//right 滑动方向 左/右 , 上/下
    speed : 2,
    speedQuick : 4,
    autoplay : false,
    lazyimg : false,
    loop : true 
});
```  


#### Object function
`go`  
`stop`  
`slidePrev`  
`slideNext`  

```javascript
myScrollSlider1.slideNext()
```
